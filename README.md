# digi-gallery
DigiGallery is a virtual art gallery for local Virginia artists to display their work to the rest of the world. Increasing accessibility to the arts, both for artists and art appreciators. 


Created originally by Elena Ceravalo, Abigail Davis, Stacy Lu, and Elizabeth Arthur for TribeHacks IX 2024, a 28 hour hackathon at the College of William & Mary.
See the project submission on DevPost: https://devpost.com/software/digigallery
